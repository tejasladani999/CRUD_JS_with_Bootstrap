#CRUD Application with JavaScript and Bootstrap
This repository contains a simple CRUD (Create, Read, Update, Delete) application built with JavaScript and Bootstrap. The application allows users to perform basic operations on a dataset using a user-friendly interface.

Features
Create: Add new items to the dataset.
Read: View the existing items in a paginated format.
Update: Modify the details of existing items.
Delete: Remove items from the dataset.

Usage
Open the index.html file in your web browser.
Use the provided interface to perform CRUD operations on the dataset.
Technologies Used
JavaScript: The primary programming language for building the CRUD functionality.
Bootstrap: Used for styling and creating a responsive user interface.
